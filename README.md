# Nelli Crafts Landing Page

A beautiful, responsive landing page for Nelli Crafts - a jewelry brand specializing in epoxy resin jewelry with Armenian flowers.

## Features

- **Modern Design**: Clean, elegant design with warm brown color scheme
- **Responsive Layout**: Works perfectly on all devices (desktop, tablet, mobile)
- **Smooth Animations**: Subtle scroll-triggered animations and hover effects
- **Contact Form**: Functional contact form with validation
- **Social Media Integration**: Telegram and Instagram links
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## File Structure

```
NelliCraftsLanding/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
├── assets/
│   ├── images/         # Product and background images
│   └── logos/          # Brand logos and icons
└── README.md           # This file
```

## Image Requirements

### Required Images

Place the following images in the `assets/images/` folder:

1. **`hero-jewelry.jpg`** - Hero section jewelry image (recommended: 800x600px)
2. **`workshop.jpg`** - About section workshop image (recommended: 600x400px)
3. **`jewelry-1.jpg`** - Floral necklace (recommended: 400x300px)
4. **`jewelry-2.jpg`** - Flower earrings (recommended: 400x300px)
5. **`jewelry-3.jpg`** - Garden bracelet (recommended: 400x300px)
6. **`jewelry-4.jpg`** - Blossom ring (recommended: 400x300px)

### Required Logos

Place the following logos in the `assets/logos/` folder:

1. **`logo.png`** - Main brand logo (recommended: 200x200px, transparent background)
2. **`telegram-icon.png`** - Telegram icon (recommended: 48x48px)
3. **`instagram-icon.png`** - Instagram icon (recommended: 48x48px)

## Customization

### Colors
The main color scheme uses warm browns (#8B4513, #A0522D). You can customize these in `styles.css`:

```css
:root {
    --primary-color: #8B4513;
    --secondary-color: #A0522D;
    --accent-color: #A0522D;
}
```

### Text Content
Update the text content in `index.html` to match your brand:
- Company name and taglines
- About section description
- Product descriptions
- Contact information

### Social Media Links
Update the social media links in `index.html`:
```html
<a href="https://t.me/YOUR_TELEGRAM" target="_blank">@YOUR_TELEGRAM</a>
<a href="https://instagram.com/YOUR_INSTAGRAM" target="_blank">@YOUR_INSTAGRAM</a>
```
